O que é Html

HTML é uma linguagem de marcação, usada no desenvolvimento de páginas web.A tradução de HTML é HyperText Markup Language.

Utilizada por muito tempo, essa linguagem vem sendo atualizado e até chegar no HTML5, que é a versão que utilizamos agora.

O que é Tag

Tag são palavras-chaves entre sinais de 'menor que'(<) e 'maior que'(>) que são responsáveis para criar e estruturar uma página em HTML.

Cada tag tem uma função, seja para aumentar ou diminuir o tamanho de uma fonte, como para escrever texto, criar links de outras páginas ou até imagens.

As tags são utlizadas apenas no HTML, mas para outros tipos de formatações na página de um site, é necessario utilizar o CSS, que veremos outro dia.

<>Listas ordenadas

Listas ordenadas, são listas que seguem uma sequência numérica. Para realizar uma lista ordenada, utilizamos a tag ol e li. A tag "ol" também é conhecida como tag Pai. A utilização dessa tag antecede a tag "li".

Sua utilização é feita da seguinte forma: sinal de menor que (<) seguido da tag ol e barra (/) seguido do sinal maior que (>). Em seguida repita o processo, trocando a tag ol pelo li.

Listas Não ordenadas

Se você não pretende enumerar sua lista, então você pode usar essa tag: (ul).

O processo é bem parecido com as demais tags. Use o sinal de menor que (<) seguido da tag ol e barra (/) seguido do sinal maior que (>).
